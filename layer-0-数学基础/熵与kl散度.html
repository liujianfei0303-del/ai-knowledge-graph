熵与KL散度

核心要点

信息熵量化随机变量的不确定性，定义为 H(p) = -∑ p(x) log p(x)，单位取决于对数底（bit / nat）。联合熵 H(X,Y)、条件熵 H(Y|X) 和互信息 I(X;Y) = H(Y) - H(Y|X) 构成信息论的完整描述体系。KL 散度（相对熵）衡量两个分布的距离：D_KL(p‖q) = ∑ p(x) log(p(x)/q(x))，非对称且非负，当且仅当 p=q 时为零。KL 散度是变分推断（ELBO 推导）、VAE、策略梯度（PPO 的 KL 惩罚）和知识蒸馏（学生分布逼近教师分布）的核心数学工具。JS 散度是 KL 的对称平滑版本，曾用于 GAN 训练。交叉熵 = 熵 + KL 散度，连接了信息论和损失函数设计。主要局限是 KL 散度在 q(x)=0 而 p(x)>0 时趋于无穷。

主要内容

信息论基础体系： 信息熵 H(p) 度量从分布 p 中采样一个事件所需的平均信息量，其对数形式源自 Shannon 信息量的可加性公理。对于离散随机变量 X，熵在均匀分布时取最大值 log|X|（不确定性最大），在退化分布时取最小值 0。联合熵 H(X,Y) = -∑∑ p(x,y) log p(x,y) 描述联合分布的不确定性。条件熵 H(Y|X) = H(X,Y) - H(X) 量化已知 X 后 Y 剩余的不确定性。互信息 I(X;Y) = H(Y) - H(Y|X) = D_KL(p(x,y)‖p(x)p(y)) 衡量两个变量共享的信息量，为零当且仅当 X 与 Y 独立。这些量通过信息论维恩图形成完整的定量框架，在特征选择（最大互信息）、聚类和因果推断中广泛应用。

KL 散度的数学性质与变体： KL 散度 D_KL(p‖q) = ∑ p(x) log(p(x)/q(x)) 满足 Gibbs 不等式（非负性）但不对称且不满足三角不等式，因此不是严格意义上的距离度量。其前向变体 D_KL(p‖q) 对 q 的覆盖性要求高（若 p(x)>0 则 q(x) 必须 >0），称零避免（zero-avoiding），在变分推断中对应”质量覆盖”模式。反向 KL D_KL(q‖p) 倾向于找到 p 的单个峰（zero-forcing），在期望传播中使用。JS 散度 JSD(p‖q) = ½D_KL(p‖(p+q)/2) + ½D_KL(q‖(p+q)/2) 是对称且有限的（范围 [0, log2]），在 GAN 训练中替代 KL 解决模式坍塌问题。Wasserstein 距离进一步克服 JS 在分布不重叠时梯度消失的缺陷。

AI 核心应用： 在 VAE 中，ELBO = E_q[log p(x|z)] - D_KL(q(z|x)‖p(z))，KL 项正则化隐变量的后验分布使其接近先验标准正态分布。在知识蒸馏中，学生网络通过最小化与学生 logits 和教师 logits 之间的 KL 散度来学习教师的行为。在强化学习中，PPO 算法在策略更新时施加 KL 惩罚项 D_KL(π_new‖π_old) ≤ δ 以防止策略更新过大导致性能崩塌。在 RLHF 中，DPO 及其变体隐式优化参考模型与对齐模型之间的 KL 散度。信息瓶颈理论（Tishby et al.）将互信息引入深度学习解释：网络各层最大化 I(Z;Y)（与任务的相关性）同时最小化 I(Z;X)（输入压缩），形成对表征学习的深层理解。

熵衡量概率分布的不确定性，KL散度度量两个概率分布之间的差异，两者是信息论和机器学习损失函数的核心概念。
