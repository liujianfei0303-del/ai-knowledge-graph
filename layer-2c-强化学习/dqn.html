DQN

Deep Q-Network（DQN）是一种将深度神经网络与 Q 学习结合的算法，通过经验回放和目标网络解决值函数近似中的不稳定问题，首次在 Atari 游戏上达到人类水平。

核心要点

DQN 将 Q-Learning 与深度神经网络结合，使用 DNN Q(s,a;θ) 作为值函数近似器以处理高维状态空间（如原始像素输入）。其训练依赖两个关键技术创新：(1) 经验回放（Experience Replay）——将 agent 与环境交互的转移样本 (s,a,r,s′) 存入回放缓冲区，训练时从中随机采样 mini-batch，打破样本间的时序相关性，提高数据效率和稳定性；(2) 目标网络（Target Network）——冻结一个独立的 Q(s,a;θ−) 网络用于生成 TD target，定期从 θ 复制权重，减少目标值随网络更新不断移动导致的训练震荡。DQN 的损失函数为 L(θ)=E[(r+γmaxa′​Q(s′,a′;θ−)−Q(s,a;θ))2]。

DQN 在 Atari 2600 游戏上取得了超越人类玩家的成绩，开创了深度强化学习时代。后续主要改进包括：Double DQN（用在线网络选动作、目标网络估值，缓解 Q 值过估计）、Dueling DQN（将 Q 值分解为状态价值 V(s) 和动作优势 A(s,a) 两路输出）、Prioritized Experience Replay（按 TD 误差大小优先采样重要样本）以及 DQN + PER + NoisyNet 的组合在 Rainbow DQN 中达到当时 SOTA。

主要内容

算法框架与训练机制

DQN 的损失函数为 L(θ)=E(s,a,r,s′)∼D​[(r+γmaxa′​Q(s′,a′;θ−)−Q(s,a;θ))2]，其中 D 为经验回放缓冲区。训练时从 D 中随机采样 mini-batch，打破了连续样本间的强相关性——这是深度学习在 RL 中成功运用的关键。目标网络 θ− 每 C 步从 θ 复制一次权重，在两次更新间保持固定，使 TD target 相对稳定，减少了值函数近似中的发散风险。DQN 的卷积神经网络架构将 Atari 游戏的四帧堆叠图像 84×84×4 映射到离散动作空间上的 Q 值输出。

主要改进与变体

DQN 衍生出大量改进变体。(1) Double DQN：使用在线网络 θ 选择动作、目标网络 θ− 评估价值，y=r+γQ(s′,argmaxa′​Q(s′,a′;θ);θ−)，有效缓解 Q 值过估计问题。(2) Dueling DQN：将 Q 值分解为状态价值 V(s;θ) 和动作优势 A(s,a;θ)，Q(s,a;θ,α,β)=V(s;θ,β)+A(s,a;θ,α)−∣A∣1​∑a′​A(s,a′;θ,α)，使网络在不影响其他动作评估的前提下学习哪些状态是有价值的。(3) Prioritized Experience Replay：按 ∣δ∣（TD error 绝对值）为每个样本分配采样优先级，提高重要样本的回放频率。(4) NoisyNet DQN：在网络参数中加入可学习噪声，实现参数空间中的高效探索。Rainbow DQN 集成上述六项改进，在 Atari 基准上达到当时 SOTA。

应用场景与影响

DQN 在 49 款 Atari 2600 游戏中以原始像素为输入取得超越人类专业玩家的成绩，被认为开辟了深度强化学习（Deep RL）这一研究领域。其影响超越游戏：DQN 的思想被应用于机器人抓取（像素→力矩）、自动驾驶感知-决策联合学习、芯片布局优化（Google’s chip placement with RL）和能源系统调度。经验回放和目标网络两大机制已成为深度 RL 算法的基础组件，被 DDPG、TD3、SAC 等后续算法继承和改进。DQN 的成功也推动了表示学习与强化学习的交叉研究——如何在 RL 中学习有效的状态表示，至今仍是核心课题。
